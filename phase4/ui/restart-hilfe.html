Wie geht Restart?

Der Stack besteht aus Router, RAG-API und UI. Nach Modell- oder API-Aenderungen die betroffenen Dienste neu starten.

Einzeldienste neu starten

sudo systemctl restart llm-router.service
sudo systemctl restart llm-rag-api.service
sudo systemctl restart llm-ui.service
sudo systemctl status llm-router.service --no-pager
sudo systemctl status llm-rag-api.service --no-pager
sudo systemctl status llm-ui.service --no-pager

Kompletter Stack

cd /home/clemi/projekte/LLM
sudo /home/clemi/projekte/LLM/phase6/install_autostart_stack.sh

Das Script installiert/aktualisiert Units und startet alle Dienste inklusive Timer.
